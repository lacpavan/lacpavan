# Índice dos Exercícios

Este arquivo resume o que existe em cada pacote para facilitar revisão, estudo e manutenção.

## JAVA_INTRO

Exercícios iniciais de entrada, saída, variáveis, operadores aritméticos e conversões simples.

| Arquivo | Tema principal |
| --- | --- |
| `Exercicio1.java` | Converte idade informada em anos, meses e dias para total aproximado em dias. |
| `Exercicio2.java` | Converte total de dias vividos para anos, meses e dias. |
| `Exercicio3.java` | Converte duração de evento em segundos para horas, minutos e segundos. |
| `Exercicio4.java` | Calcula expressões matemáticas usando `Math.pow`. |

## Java_LacosDEC

Exercícios de tomada de decisão com `if`, `else if`, `else` e `switch`.

| Arquivo | Tema principal |
| --- | --- |
| `LacosDec_exerc_1.java` | Identifica o maior entre três números. |
| `LacosDec_exerc_2.java` | Ordena três números em ordem crescente. |
| `LacosDec_exerc_3.java` | Classifica idade ou perfil conforme as regras do exercício. |
| `LacosDec_exerc_4.java` | Aplica regra condicional a partir de entradas do usuário. |

## Java_LacosRep

Exercícios de repetição com `for`, `while` e `do while`.

| Arquivo | Tema principal |
| --- | --- |
| `LacosRep_exFor_1.java` | Repetição contada com `for`. |
| `LacosRep_exFor_2.java` | Acumuladores e contadores dentro de `for`. |
| `LacosRep_exWhile_1.java` | Entrada repetida com condição de parada. |
| `LacosRep_exWhile_2.java` | Pesquisa com contadores por idade, sexo e temperamento. |
| `LacosRep_exDoWhile_1.java` | Menu ou leitura repetida com execução mínima garantida. |
| `LacosRep_exDoWhile_2.java` | Repetição com validação usando `do while`. |

## Arrays

Exercícios de vetores e matrizes.

| Arquivo | Tema principal |
| --- | --- |
| `Array_ex1.java` | Vetor de inteiros, soma de posições e alteração de valores. |
| `Array_ex3.java` | Matriz 3x3 e contagem de valores maiores que 10. |

## POO

Primeiros exercícios de orientação a objetos.

| Arquivo | Tema principal |
| --- | --- |
| `Cliente.java` | Classe com atributos, construtor, getters, setters e impressão de dados. |
| `TesteCliente.java` | Cria e testa objeto da classe `Cliente`. |
| `Aviao.java` | Classe com atributos e métodos para representar um avião. |
| `TesteAviao.java` | Cria e testa objeto da classe `Aviao`. |
